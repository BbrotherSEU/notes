
# Web FrontEnd Note

## reactJS

 - create project

```
npm install -g create-react-app
create-react-app my-app
```

 - run project

```
npm start
http://localhost:3000
```

## Ant.design

 - after you create reactJS app, run :

```
yarn add antd
```

 - use antd
    - - 修改 src/App.css，在文件顶部引入 antd/dist/antd.css。

```
@import '~antd/dist/antd.css';
```
